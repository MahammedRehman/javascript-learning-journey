# javascript-learning-journey

### Tips 

If you want to keep the last digit(s) then use modulo(%).<br>
Example: 1234 % 10 -> 4 or 1234 % 100 -> 34

If you want to remove the last digit(s) then use divide(/).<br>
Example: 1234 / 10 -> 123 or 1234 / 100 -> 12
use Math.floor() for accurate values

#### Function Expression
In React, almost every component and handler is a function expression or arrow function. You'll almost never use function declarations in modern frontend code. But you must understand hoisting for interviews.
